# Bright-coffe-shop-
**OVERVIEW**
This project analyzes transactional sales data from Bright Coffee Shop to uncover actionable insights that support business growth and decision-making.The objective was to transform raw sales data into meaningful insights for the CEO, focusing on revenue optimization and product performance.
**OBJECTIVE**
Identify top-performing products by revenue,
Analyze sales performance across different times of day,
Evaluate product trends and customer purchasing behavior,
Provide data-driven recommendations to improve sales.
**TOOLS AT DISPOSAL**
Data Processing:Databricks, SQL
Data Analysis:Microsoft Excel 
Visualization:Excel
Planning & Architecture:Miro
Presentation:Canva
**Data Processing Workflow**
Converted raw Excel dataset to CSV format
Loaded data into Databricks
Performed transformations:
Cleaned unit_price values 
Created transaction_time_bucket 
Calculated total_amount = unit_price × transaction_qty
Aggregated data using SQL for analysis
**CONCLUSION**
This project showcases the end-to-end data analysis process—from raw data preparation to delivering actionable business insights. Through structured data processing and visualization, key trends in sales performance, product demand, and customer behavior were identified.

The findings provide clear opportunities for improving revenue through strategic decision-making, including optimizing product offerings, targeting low-sales periods, and aligning operations with peak demand.
